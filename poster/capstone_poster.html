ğŸ“„ Abstract

This capstone project presents a comprehensive deep learning solution for detecting and classifying Bangladeshi traffic signs using YOLOv11. We trained on the Bangladeshi Road Sign Detection Dataset (BRSDD) containing 8,953 images across 29 sign categories. Our YOLOv11 nano model achieved exceptional performance with 99.45% mAP@50, demonstrating near-perfect detection capability. The project includes complete deployment solutions: a real-time Android mobile application and a web-based demo interface.

Keywords: Traffic Sign Detection, YOLOv11, Object Detection, Deep Learning, Computer Vision, Bangladesh, Road Safety
