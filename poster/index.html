Field Context Why this matters

Bangladesh roads mix dense traffic, occlusions, and uneven lighting. Reliable sign detection is a prerequisite for ADAS and smart mobility.

  • Real-time perception for low-cost dashcams and mobile devices.
  • Localized dataset capturing non-standard signs and weather noise.
  • Benchmarking modern one-stage detectors for edge deployment.

Pipeline

Acquire + Annotate Field captures + YOLO labels.
Preprocess Resize, normalize, stratified splits.
Train YOLOv11n fine-tune (COCO init) on CUDA/CPU fallback.
Validate mAP@50, precision/recall, FPS on GPU & CPU.
Deploy Android app + web demo using exported weights.

Takeaways

  • YOLOv11n is the sweet spot for accuracy vs latency on edge devices.
  • Data preprocessing script outputs YOLO-ready splits and data.yaml.
  • Evaluation script compares detectors with consistent metrics.
  • Extend dataset with night/rain captures for harder scenarios.
  • Quantize weights for microcontrollers and low-power boards.
  • Integrate SSD loader to finalize the secondary baseline.
